## 功能特征

- 提供统一API，支持多种开源交易机器人或商业闭源机器人.
- 支持所有客户端，任意时间，任意地点管理您的交易.
- 支持管理多个机器人，一键切换
- 支持实时交易，支持多空操作，支持买卖点显示
- 支持配置参数实时修改
- 支持策略参数实时修改

## 安装
